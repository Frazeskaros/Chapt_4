επιλεχθηκε η 2η ασκηση "αναλυση ενος log file στην επιθυμητη μορφη και ευρεση ολων των φορων που εμφανιζεται μια συγκεκριμενη λεξη"

Για να τρεχει το script θα χρησιμοποιησετε στο terminal την εντολη
./findword.sh arg1

οπου arg1 η λεξη που θελουμε να βρουμε στο Log file (file.txt)

Η μετρηση γινεται με το command "grep" οπου βρισκει pattern σε αρχεια.
με το -o δινουμε εντολη στο grep εαν βρει παραπανω απο μια φορα την λεξη σε μια γραμμη να την μετρησει ως 2+. 
με το -w δινουμε εντολη στο grep ποια λεξη να ψαξει.
τελος με το wc -l μετραμε ποσες φορες βρεθηκε η λεξη που θελουμε.
